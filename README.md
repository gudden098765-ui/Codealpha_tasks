# CodeAlpha Python Internship Projects 🐍

This repository contains Python projects completed as part of the CodeAlpha Python Internship.

## 📌 Projects Included

## 1. 🎮 Hangman Game

### Description
A simple text-based Hangman game where the player guesses a hidden word one letter at a time.

### Features
- Random word selection
- 6 incorrect attempts limit
- Console-based gameplay

### Concepts Used
- random module
- while loop
- if-else
- strings
- lists

## 2. 📈 Stock Portfolio Tracker

### Description
A Python program that calculates the total investment value based on user-entered stock names and quantities.

### Features
- User inputs stock name and quantity
- Uses predefined stock prices
- Calculates total investment

### Concepts Used
- Dictionary
- User input/output
- Arithmetic operations
- Basic Python logic

## 3. 🤖 Basic Chatbot

### Concepts Used

- input/output
